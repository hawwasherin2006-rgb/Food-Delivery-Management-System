# Food-Delivery-Management-System
Designed normalized Database schema to track customer orders, restaurants and delivery agents

## Tools Used
- MySQL
- Excel
- Draw.io

## What I did
- Designed a custome dataset representing customers, restaurants, orders and deliveries and payments
- Cleaned and normalized the data to remove redundancy and ensure consistency.
- Designed the database schema and relationships.
- Created Entity-Relationship (ER) diagrams to model the system.
- Implemented the schema in MySQL.
- Loaded the cleaned data into database tables.
- Wrote and executed SQL queries to retrieve insights and reports.
- Tested queries for accuracy and performance.

## Outcome

Developed a fully normalized MySQL database with ER diagrams and analytical SQL queries to support food delivery operations and reporting.
